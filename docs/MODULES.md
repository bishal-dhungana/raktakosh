# Module Catalogue

| Module | Main users | Primary functions |
|---|---|---|
| Public discovery | Guests and requesters | Search verified facility availability across all 77 Nepal districts by group, Rh factor, and component. |
| Request coordination | Requesters | Submit and track a blood coordination request, attach a supporting document, and read the status timeline. |
| Donor management | Donors | Maintain outreach consent, availability state, contact preferences, campaign responses, private date of birth, derived age, and consented pre-screening answers. |
| Facility inventory | Inventory managers and facility administrators | Record availability, reserved quantity, public visibility, adjustment reason, and update history. |
| Request review | Reviewers and facility administrators | Review queued requests, add internal notes, change only to permitted workflow states, and launch eligible outreach. |
| Blood-centre operations | Verified facility staff | Use a role-scoped operations dashboard. Reviewers and facility administrators can access only their facility's active requests and consented donor responses; inventory managers retain inventory-only access. |
| Donor outreach | Reviewers and donors | Select consented eligible donors, issue a privacy-minimised invitation, and collect responses. |
| Governance and audit | Platform administrators | Review facility status, policy versions, and high-signal audit events. |
| Localization and usability | All users | English/Nepali public content, keyboard-visible focus states, responsive layouts, and local-time display. |
